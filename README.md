# 🎬 MoodMatch

MoodMatch is a full-stack movie recommendation web application that suggests movies based on the user's mood and tracks user activity with social and admin features.

---

## 🚀 Features

- 🎭 Mood-based movie discovery
- 🔐 User authentication (login/signup)
- ❤️ Watch history tracking
- 🌐 Social sync feed
- 🛠️ Admin dashboard (view, deactivate, delete users)
- 📱 Responsive modern UI

---

## 🧠 Tech Stack

**Frontend**
- HTML5
- CSS3
- JavaScript

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB (Mongoose)

**API**
- TMDB Movie API
